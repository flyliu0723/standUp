# 顶部健康 HUD（Status Bar）

尺寸：横向 480 × 40 px；竖向 40 × 480 px。圆角胶囊。

## 位置设置

- 单显示器：四边均可选（上/下/左/右）
- 多显示器：自动排除拼接内边（如双屏横排时隐藏两屏中间的左右边），最多 6 个外缘位置
- 左/右侧自动切换竖向布局
- 设置路径：设置 → 桌面与显示 → 状态条位置

## 信息布局

```
┌─────────────────────────────────────────────────────────────┐
│ ████████████░░░░░░░░░░░░░░░░░░  ← Body 肉体线（顶部 2px）   │
│ ～～12  [呼吸圈]  坐姿时间 · 保持专注        04:52 · 换脑拉伸 │
│  ^Mind波形+数值                                               │
└─────────────────────────────────────────────────────────────┘
```

| 模块 | 元素 | 逻辑 |
|------|------|------|
| Body | 顶部 2px 渐变条 | 久坐进度 0→1，绿→黄→橙 |
| Mind | SVG 折线 + 0–100 数字 | 近 8 分钟切窗频率波形 + 精神负载分（竖向时整块内容旋转 90°，波形仍横向展示） |
| 行动 | 主文案 + 副文案 | 如 `04:52 · 换脑拉伸` |
| 呼吸 | 双环动画 | 仅 `breathing` 模式显示，4:7 节奏（11s 周期） |

## 模式与配色

| mode | 触发条件 | 视觉 |
|------|----------|------|
| `normal` | 默认 | 深灰底，黄倒计时 |
| `mental_oom` | 5min 切窗≥30 且 应用≥3 | 雾蓝底，波形抖动 |
| `breathing` | 键盘停 + 等待间隙 5–60s | 紫调底 + 呼吸光圈 |
| `phase_break` | 高切换后静止≥60s | 暗橘底 + 微光脉冲 |
| `off_duty` | 未上班 | 中性灰 |

## 数据流

